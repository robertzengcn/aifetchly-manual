# Google Maps 抓取工具

aiFetchly 的 Google Maps 抓取工具允许您通过关键词和位置在 Google Maps 上搜索本地商家。您可以收集全面的商家数据，包括名称、类别、评分、评论、地址、电话号码和网站 URL — 所有信息均来自 Google Maps 搜索结果。

## 概述

Google Maps 抓取工具提供了一个简洁的界面，用于提取本地商家数据：

1. **输入关键词**（例如"牙医"、"意大利餐厅"）
2. **输入位置**（例如"纽约"、"伦敦"、"90210"）
3. **配置选项**（最大结果数、是否包含网站、评论等）
4. **开始搜索**并实时监控进度
5. **导出结果**为 CSV 或 JSON 格式

:::tip 最佳用途

Google Maps 抓取工具非常适合查找带有结构化数据的本地商家，包括经过验证的地址、电话号码、评分和营业时间 — 这些信息比一般的网络搜索结果更加可靠。

:::

## 访问 Google Maps 抓取工具

1. 点击左侧导航菜单中的 **Google Maps**
2. 抓取工具打开后包含两个标签页：**搜索** 和 **历史记录**

## 运行搜索

### 第一步：输入搜索条件

#### 商家关键词（必填）

输入要搜索的商家类型或名称：

- **示例**：`dentist`、`Italian restaurant`、`plumber`、`marketing agency`
- **提示**：使用具体的商家类型可获得更精准的结果

#### 位置（必填）

输入城市、地址或邮政编码：

- **示例**：`New York`、`London`、`90210`、`Paris, France`
- **提示**：更精确的位置会产生更相关的结果

### 第二步：配置搜索选项

#### 最大结果数

- **范围**：1–50 条结果
- **默认值**：20 条结果
- 调整滑块以控制要提取的商家数量

#### 包含网站

- **启用**（默认）：尝试提取商家网站 URL
- **禁用**：跳过网站提取以加快速度

#### 包含评论

- **禁用**（默认）：仅返回基本商家数据
- **启用**：在结果中包含评论数据（会增加处理时间）

#### 显示浏览器

- **禁用**（默认）：浏览器以无头模式运行（速度更快）
- **启用**：抓取过程中浏览器窗口可见（适用于调试）

:::warning 显示浏览器

启用此选项将在搜索过程中在您的屏幕上显示浏览器窗口。此功能仅用于调试目的，可能会减慢抓取速度。

:::

### 第三步：账号和代理设置（可选）

#### Google 账号

选择一个 Google 账号以使用其 Cookie 进行经过身份验证的抓取：

- **优势**：
  - 更高的成功率
  - 获取更详细的商家信息
  - 降低被屏蔽的风险
- 下拉菜单中仅列出 Google 账号

:::tip 添加账号

要添加 Google 账号，请前往 **设置** 并配置您的社交账号。详情请参阅[系统设置](../settings/system-settings)。

:::

#### 代理

选择一个或多个代理以在搜索过程中轮流使用：

- **优势**：
  - 将请求分配到多个 IP
  - 降低被检测的风险
  - 大规模抓取的必备条件
- 选择多个代理可按请求自动轮换

### 第四步：开始搜索

1. 点击 **开始搜索** 开始
2. 出现一个进度条，显示实时抓取状态
3. 搜索异步运行 — 您可以继续使用其他功能
4. 最多可同时运行 **3 个并发搜索**

:::info 并发搜索

您最多可以同时运行 3 个 Google Maps 搜索。如果您尝试启动第 4 个，系统会提示您等待其中一个完成。

:::

### 取消搜索

如果搜索时间过长或您想停止：

- 点击搜索运行时出现的 **取消** 按钮
- 取消前已收集的部分结果将被保留

## 查看结果

搜索完成后，结果将以数据表格的形式显示。

### 结果表格

| 列名 | 描述 |
|------|------|
| **名称** | 商家名称（如果可用，可点击链接到 Google Maps） |
| **类别** | 商家类别（例如"餐厅"、"牙医"） |
| **评分** | 星级评分（带星标图标） |
| **评论** | 评论数量 |
| **地址** | 完整商家地址 |
| **电话** | 电话号码 |
| **网站** | 商家网站的可点击链接（截断显示） |

:::tip 点击商家名称

带有 Google Maps 链接的商家名称是可点击的 — 它们会在新标签页中打开 Google Maps 上的商家页面。

:::

### 搜索结果标题

标题显示：

- 找到的商家总数
- 使用的搜索关键词
- 搜索的位置

### 无结果

如果没有找到商家，会显示一条消息，表示您的搜索条件没有找到结果。请尝试：

- 使用更广泛的关键词
- 扩大位置范围
- 增加最大结果数设置

## 导出结果

### 导出为 CSV

1. 点击结果标题中的 **导出 CSV**
2. CSV 文件会自动下载
3. 文件名包含关键词和位置（例如 `google-maps-dentist-NewYork.csv`）

**CSV 列包括：**
名称、类别、评分、评论数量、地址、电话、网站、营业时间、地图 URL

### 导出为 JSON

1. 点击结果标题中的 **导出 JSON**
2. JSON 文件会下载，包含完整的结果数据
3. 文件名遵循与 CSV 导出相同的命名规则

## 搜索历史

**历史记录** 标签页存储您所有过去的 Google Maps 搜索。

### 查看历史记录

1. 点击 **历史记录** 标签页
2. 表格显示所有以前的搜索

| 列名 | 描述 |
|------|------|
| **查询** | 使用的搜索关键词 |
| **位置** | 搜索的位置 |
| **结果** | 找到的商家总数 |
| **日期** | 执行搜索的时间 |
| **操作** | 查看或删除 |

### 历史操作

- **查看**（眼睛图标）：将历史结果加载到搜索标签页中
- **删除**（垃圾桶图标）：永久删除搜索记录

### 刷新历史记录

- 点击 **刷新** 按钮重新加载历史记录列表
- 点击历史记录标签页时会自动加载历史记录

## 最佳实践

### 1. 关键词策略

- **具体明确**：使用精确的商家类型
  - ✅ `Italian restaurant`
  - ❌ `food`
- **尝试变体**：搜索多个词以获得全面覆盖
  - `dentist` 和 `dental clinic`
  - `plumber` 和 `plumbing service`

### 2. 位置定位

- **使用具体位置** 以获得更好的结果：
  - ✅ `Manhattan, New York`
  - ✅ `90210`
  - ❌ `USA`
- **搜索多个区域** 以覆盖更广的范围：
  - 对每个社区或区域分别运行搜索

### 3. 最大结果数设置

- **快速测试**：设置为 5–10 条结果以验证您的搜索
- **标准搜索**：20 条结果（默认）
- **全面搜索**：50 条结果以获取彻底的覆盖

### 4. 使用代理

- **单次搜索**：不需要代理
- **连续多次搜索**：使用 1–2 个代理
- **大规模抓取**：使用 3 个以上代理并开启轮换

### 5. 使用 Google 账号

- **推荐**用于返回 30 条以上结果的搜索
- **必备**当抓取商家密集的热门区域时
- 有活跃 Google Maps 使用记录的账号效果更好

## 故障排除

### 搜索失败

**可能的原因：**
- 网络连接问题
- Google Maps 屏蔽了请求
- 关键词或位置无效

**解决方案：**
1. 检查您的网络连接
2. 尝试不同的关键词或位置
3. 使用 Google 账号进行经过身份验证的访问
4. 启用代理进行 IP 轮换
5. 降低最大结果数设置

### 未找到结果

**可能的原因：**
- 关键词与该区域的任何商家都不匹配
- 位置过于具体或过于模糊
- Google Maps 返回了空结果

**解决方案：**
1. 尝试更广泛的关键词（例如用 `restaurant` 代替 `Italian restaurant`）
2. 使用更大的城市或更广的区域
3. 在 Google Maps 上手动验证该商家类型在该区域是否存在

### 结果不完整

**可能的原因：**
- 搜索在完成前被取消
- 部分商家列表缺少所需数据
- 抓取过程中触发了频率限制

**解决方案：**
1. 让搜索完全完成
2. 对剩余区域运行另一次搜索
3. 使用代理以避免频率限制
