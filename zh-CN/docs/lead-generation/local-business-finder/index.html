本地商家查找工具

aiFetchly 的本地商家查找工具允许您使用两种数据来源按关键词和位置搜索本地商家：Google Maps（全球覆盖）和 Yandex Maps（适用于俄罗斯和独联体市场）。收集全面的商家数据，包括名称、类别、评分、评论、地址、电话号码和网站 URL。

概述​

本地商家查找工具提供了一个简洁的界面，用于整理本地商家数据：

  1. 选择数据来源 — Google Maps 或 Yandex Maps
  2. 输入关键词（例如"牙医"、"意大利餐厅"）
  3. 输入位置（例如"纽约"、"莫斯科"、"90210"）
  4. 配置选项（最大结果数、是否包含网站、评论等）
  5. 开始搜索并实时监控进度
  6. 导出结果为 CSV、JSON 格式或复制到剪贴板
选择数据来源
  • Google Maps — 适用于全球市场和西方国家。在全球范围内覆盖更优。
  • Yandex Maps — 俄罗斯、独联体国家（哈萨克斯坦、白俄罗斯）和土耳其的必备选择。支持语言和地区自定义。

如需对某个区域进行全面覆盖，请在两个数据来源上分别运行搜索，然后交叉对比并去除重复结果。

访问本地商家查找工具​

  1. 点击左侧导航菜单中的 Local Business Finder
  2. 页面打开后包含两个标签页：搜索 和 历史记录

运行搜索​

第一步：选择数据来源​

在搜索表单顶部选择您偏好的数据来源：

特性Google MapsYandex Maps
最佳用途全球市场、西方国家俄罗斯、独联体、土耳其
语言支持多语言（自动）可配置语言/地区
商家覆盖全球最佳俄罗斯/独联体最佳
账号类型Google 账号Yandex 账号

第二步：输入搜索条件​

商家关键词（必填）​

输入要搜索的商家类型或名称：

  • 示例：dentist、Italian restaurant、plumber、marketing agency
  • 提示：使用具体的商家类型可获得更精准的结果
  • Yandex 提示：使用当地语言的关键词效果最佳（例如在俄罗斯地区使用俄语关键词）

位置（必填）​

输入城市、地址或邮政编码：

  • 示例：New York、London、90210、Moscow、Saint Petersburg、Almaty, Kazakhstan
  • 提示：更精确的位置会产生更相关的结果

第三步：配置搜索选项​

最大结果数​

  • 范围：1–50 条结果
  • 默认值：20 条结果
  • 调整滑块以控制要包含的商家数量
结果上限

最大结果数设有安全上限，以确保可靠的用户资料洞察。滑块范围 1–50 是用户可配置的范围。

包含网站​

  • 启用（默认）：尝试包含商家网站 URL
  • 禁用：跳过网站查找以加快速度

包含评论​

  • 禁用（默认）：仅返回基本商家数据
  • 启用：在结果中包含评论数据（会增加处理时间）

显示浏览器​

  • 禁用（默认）：浏览器以无头模式运行（速度更快）
  • 启用：信息整理过程中浏览器窗口可见（适用于调试）
显示浏览器

启用此选项将在搜索过程中在您的屏幕上显示浏览器窗口。此功能仅用于调试目的，可能会减慢信息整理过程。

第四步：语言和地区设置（仅限 Yandex Maps）​

这些设置仅适用于 Yandex Maps，可帮助自定义您的搜索上下文。

语言​

  • 设置 Yandex Maps 的界面语言代码
  • 示例：ru（俄语）、en（英语）、tr（土耳其语）
  • 留空则使用默认语言
  • 提示：使用当地语言可在该地区获得更好的搜索结果

地区​

  • 设置搜索上下文的地区代码
  • 示例：ru（俄罗斯）、kz（哈萨克斯坦）、by（白俄罗斯）
  • 留空则使用默认地区
  • 提示：设置正确的地区可提高结果准确性

第五步：账号和代理设置（可选）​

Google 账号 / Yandex 账号​

选择一个账号以使用其 Cookie 进行经过身份验证的信息整理：

  • 优势：
    • 更高的成功率
    • 获取更详细的商家信息
    • 降低被屏蔽的风险
  • Google Maps 仅列出 Google 账号；Yandex Maps 仅列出 Yandex 账号
添加账号

要添加账号，请前往 设置 并配置您的社交账号。详情请参阅系统设置。

代理​

选择一个或多个代理以在搜索过程中轮流使用：

  • 优势：
    • 将请求分配到多个 IP
    • 降低被检测的风险
    • 大规模公共信息整理的必备条件
  • 选择多个代理可按请求自动轮换
代理建议

对于 Yandex Maps，使用位于目标地区的代理（例如搜索莫斯科时使用俄罗斯代理）可显著提高成功率。

第六步：开始搜索​

  1. 点击 开始搜索 开始
  2. 出现一个进度条，显示实时进度状态
  3. 搜索异步运行 — 您可以继续使用其他功能
  4. 最多可同时运行 3 个并发搜索
并发搜索

您最多可以同时运行 3 个搜索。如果您尝试启动第 4 个，系统会提示您等待其中一个完成。

取消搜索​

如果搜索时间过长或您想停止：

  • 点击搜索运行时出现的 取消 按钮
  • 取消前已收集的部分结果将被保留

查看结果​

搜索完成后，结果将以数据表格的形式显示。

结果表格​

列名描述
名称商家名称（如果可用，可点击链接到地图）
类别商家类别（例如"餐厅"、"牙医"）
评分星级评分（带星标图标）
评论评论数量
地址完整商家地址
电话电话号码
网站商家网站的可点击链接（截断显示）
点击商家名称

带有地图链接的商家名称是可点击的 — 它们会在新标签页中打开地图上的商家页面。

搜索结果标题​

标题显示：

  • 找到的商家总数
  • 使用的搜索关键词
  • 搜索的位置

无结果​

如果没有找到商家，会显示一条消息，表示您的搜索条件没有找到结果。请尝试：

  • 使用更广泛的关键词
  • 使用当地语言的关键词（适用于 Yandex Maps）
  • 扩大位置范围
  • 增加最大结果数设置

导出结果​

复制全部​

  1. 点击结果标题中的 复制全部
  2. 所有结果将以 JSON 格式复制到剪贴板
  3. 粘贴到任何文本编辑器或电子表格中

导出为 CSV​

  1. 点击结果标题中的 导出 CSV
  2. CSV 文件会自动下载
  3. 文件名包含关键词和位置（例如 maps-dentist-NewYork.csv）

CSV 列包括： 名称、类别、评分、评论数量、地址、电话、网站、营业时间、地图 URL

导出为 JSON​

  1. 点击结果标题中的 导出 JSON
  2. JSON 文件会下载，包含完整的结果数据
  3. 文件名遵循与 CSV 导出相同的命名规则

搜索历史​

历史记录 标签页存储您所有过去的搜索。

查看历史记录​

  1. 点击 历史记录 标签页
  2. 表格显示所有以前的搜索
列名描述
查询使用的搜索关键词
位置搜索的位置
结果找到的商家总数
日期执行搜索的时间
操作查看或删除

历史操作​

  • 查看（眼睛图标）：将历史结果加载到搜索标签页中
  • 删除（垃圾桶图标）：永久删除搜索记录

刷新历史记录​

  • 点击 刷新 按钮重新加载历史记录列表
  • 点击历史记录标签页时会自动加载历史记录

最佳实践​

1. 关键词策略​

  • 具体明确：使用精确的商家类型
    • ✅ Italian restaurant
    • ❌ food
  • 尝试变体：搜索多个词以获得全面覆盖
    • dentist 和 dental clinic
    • plumber 和 plumbing service
  • Yandex 使用当地语言：在俄罗斯地区使用俄语关键词
    • ✅ стоматолог（俄语的牙医）用于莫斯科
    • ✅ dentist 用于英语地区

2. 位置定位​

  • 使用具体位置 以获得更好的结果：
    • ✅ Manhattan, New York
    • ✅ Moscow
    • ✅ 90210
    • ❌ USA 或 Russia（过于宽泛）
  • 搜索多个区域 以覆盖更广的范围：
    • 对每个社区或区域分别运行搜索

3. 最大结果数设置​

  • 快速测试：设置为 5–10 条结果以验证您的搜索
  • 标准搜索：20 条结果（默认）
  • 全面搜索：50 条结果以获取彻底的覆盖

4. 语言和地区设置（Yandex Maps）​

  • 将语言设置为 ru 用于俄语地区
  • 将地区设置为目标国家 以获得最佳结果
  • 如果不确定则留空 — Yandex 将根据位置使用默认值

5. 使用代理​

  • 单次搜索：不需要代理
  • 连续多次搜索：使用 1–2 个代理
  • 大规模信息整理：使用 3 个以上代理并开启轮换
  • Yandex Maps：使用当地代理（搜索俄罗斯时使用俄罗斯代理）

6. 使用账号​

  • Google 账号：推荐用于返回 30 条以上结果的搜索；商家密集的热门区域必备
  • Yandex 账号：推荐用于所有 Yandex Maps 搜索；商家密集的俄罗斯城市必备
  • 有活跃使用记录的账号效果更好

故障排除​

搜索失败​

可能的原因：

  • 网络连接问题
  • 地图服务屏蔽了请求
  • 关键词或位置无效

解决方案：

  1. 检查您的网络连接
  2. 尝试不同的关键词或位置
  3. 使用账号进行经过身份验证的访问
  4. 启用代理 — 最好使用目标地区的 IP
  5. 降低最大结果数设置
  6. （Yandex Maps）尝试设置正确的语言和地区代码

未找到结果​

可能的原因：

  • 关键词与该区域的任何商家都不匹配
  • 位置过于具体或过于模糊
  • 语言或地区设置错误（Yandex Maps）

解决方案：

  1. 尝试更广泛的关键词或使用当地语言的关键词
  2. 使用更大的城市或更广的区域
  3. 在地图服务上手动验证该商家类型在该区域是否存在
  4. （Yandex Maps）检查您的语言和地区设置

结果不完整​

可能的原因：

  • 搜索在完成前被取消
  • 部分商家列表缺少所需数据
  • 信息整理过程中触发了频率限制

解决方案：

  1. 让搜索完全完成
  2. 对剩余区域运行另一次搜索
  3. 使用代理以避免频率限制

与其他功能的集成​

本地商家查找工具的结果可用于：

  • 联系方式提取 — 从结果中找到的商家网站提取联系信息
  • 黄页目录助手 — 与目录列表交叉对比以获取更全面的数据
  • AI 邮件撰写 — 使用收集的商家数据创建个性化的推广邮件
  • 推广活动 — 使用提取的联系人信息发起邮件营销活动
